# 2417번: 정수 제곱근

### 구분

<p>Binary Search</p>

### 설명

<p>정수가 주어지면, 그 수의 정수 제곱근을 구하는 프로그램을 작성하시오.</p>

### 입력

<p>첫째 줄에 정수 n이 주어진다. (0 ≤ n < 2<sup>63</sup>)</p>

### 출력

<p>첫째 줄에 q<sup>2</sup> ≥ n인 가장 작은 음이 아닌 정수 q를 출력한다.</p>

### 예시 입력 1

<pre>122333444455555</pre>

### 예시 출력 1

<pre>11060446</pre>
